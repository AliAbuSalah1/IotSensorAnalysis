# IoT Sensor Analysis 🌡️

Reading temperature and humidity data from a DHT22 sensor
via ESP32 and analyzing it with Python and ML forecasting.

## Components
- ESP32
- DHT22 Temperature & Humidity Sensor
- Wokwi Simulator

## Files
- `main.py` — MicroPython code running on ESP32
- `data.csv` — Data collected from the sensor
- `analysis.ipynb` — Data analysis and forecasting

## Results
- Average Temperature: 24.52°C
- Standard Deviation: 1.69°C
- Next Reading Forecast: 24.70°C ± 1.40

## Tech Stack
Python | pandas | matplotlib | MicroPython | ESP32
